# 🚗 Smart Car Parking System

A web-based parking management system developed using HTML, CSS, and JavaScript that automates parking slot allocation, vehicle entry/exit tracking, and real-time parking availability monitoring.

## Features
- Parking slot allocation
- Vehicle entry and exit tracking
- Real-time parking status monitoring
- Interactive dashboard
- IoT sensor simulation
- Responsive user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Type
Frontend Web Application
